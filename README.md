# Square-Off-JS

## Description

## Run the project

```bash
npm install

live-server
```

## Run the Docker container

```bash
docker build -t square-off-js-app .

docker run -p 8080:3000 square-off-js-app
```
OR

```bash
docker-compose up --build
```

If you don't need to rebuild the image, you can just run:

```bash
docker-compose up
```
